# Outsource vs In-House Decision by Function

## Summary

This framework helps determine which business functions to outsource vs keep in-house based on function type, strategic importance, company stage, and cost thresholds. The default rule is: outsource commodity functions where external providers have scale advantages, retain functions that directly create competitive differentiation. In 2025-2026, talent access has overtaken cost savings as the primary outsourcing driver (42% vs 34%), and hybrid models — small internal core team plus external execution capacity — have become the dominant pattern for engineering, support, and marketing. [src3]

## Constraints
<!-- Agents: read before walking through this decision with a user. -->

- Fully-loaded internal costs (salary + 25-40% benefits + 15-25% overhead + management time) must be calculated before comparing to outsourcing quotes — sticker-price comparisons are meaningless
- Outsourcing core-differentiating work is effectively irreversible — once internal capability atrophies, rebuilding takes 2-3x the original development time
- Regulated functions (financial controls, data privacy, healthcare compliance) may have legal outsourcing restrictions by jurisdiction
- Quality and IP risk differ dramatically by function — engineering IP leakage is existential, payroll processing errors are recoverable
- Switching costs compound over time — vendor lock-in makes the first-year cost irrelevant if years 3-5 are trapped

## Decision Inputs

<!-- Structured list of what the agent must gather from the user before
     traversing the decision tree. Each input directly maps to a branch. -->

| Input | Why It Matters | How to Assess |
|-------|---------------|---------------|
| Function type | Each function has different outsourcing maturity and risk profiles | Classify as engineering, support, finance, HR, marketing, IT, or legal |
| Strategic importance | Core differentiators should almost never be outsourced | Apply Prahalad/Hamel test: does it create customer value, is it hard to replicate, does it enable multiple markets? |
| Company stage | Startups outsource for speed; enterprises outsource for scale | Headcount and annual revenue determine which functions justify full-time hires |
| Current internal capability | Weak internal teams may benefit from outsourcing; strong ones rarely should | Assess: do you have a senior leader in this function? Is the team performing above industry benchmarks? |
| Budget and cost structure | OpEx vs CapEx preference and absolute budget determine feasibility | Compare fully-loaded internal cost vs outsourced cost including management overhead |

## Decision Tree

```
START — Should this function be outsourced or kept in-house?
├── Is this function a core competitive differentiator?
│   ├── YES → KEEP IN-HOUSE
│   │   Reason: Outsourcing core capabilities erodes competitive advantage
│   │   Exception: Augment with contractors for surge capacity only
│   └── NO → Continue evaluation
│       ├── What function type?
│       │   ├── ENGINEERING/PRODUCT
│       │   │   ├── Core product architecture? → IN-HOUSE (always)
│       │   │   ├── Feature development at scale? → HYBRID (internal architects + external devs)
│       │   │   │   Cost: Internal $150-200K/yr fully loaded vs offshore $35-80K/yr
│       │   │   └── QA, DevOps, maintenance? → OUTSOURCE candidate
│       │   │       Cost: Offshore $25-50/hr vs internal $120-180K/yr
│       │   ├── CUSTOMER SUPPORT
│       │   │   ├── <50 tickets/day? → IN-HOUSE (Tier 1+2)
│       │   │   ├── 50-500 tickets/day? → HYBRID (outsource Tier 1, retain Tier 2+3)
│       │   │   │   Cost: Outsourced $2-8/ticket vs internal $15-25/ticket
│       │   │   └── >500 tickets/day? → OUTSOURCE Tier 1, consider Tier 2
│       │   │       Cost: BPO $8-16/hr offshore, $28-40/hr US-based
│       │   ├── FINANCE/ACCOUNTING
│       │   │   ├── Bookkeeping, AP/AR, payroll? → OUTSOURCE
│       │   │   │   Cost: $2-5K/mo vs internal $80-120K/yr + benefits
│       │   │   ├── FP&A, treasury, strategic finance? → IN-HOUSE
│       │   │   └── Tax, audit prep? → OUTSOURCE to specialized firms
│       │   ├── HR/PEOPLE OPS
│       │   │   ├── Payroll, benefits admin, compliance? → OUTSOURCE
│       │   │   │   Cost: PEO $50-150/employee/mo vs internal team $100-200K/yr
│       │   │   ├── Recruiting (volume)? → OUTSOURCE (RPO)
│       │   │   │   Cost: 15-25% of first-year salary vs internal $80-120K/yr per recruiter
│       │   │   └── Culture, L&D, employee relations? → IN-HOUSE (always)
│       │   ├── MARKETING/CREATIVE
│       │   │   ├── Brand strategy, positioning? → IN-HOUSE
│       │   │   ├── Content production, design? → HYBRID or OUTSOURCE
│       │   │   │   Cost: Agency $150-300/hr vs freelance $50-150/hr vs internal $90-140K/yr
│       │   │   └── Paid media, SEO execution? → OUTSOURCE (agency)
│       │   │       Cost: Agency retainer $3-15K/mo vs internal $80-130K/yr + tools
│       │   └── IT/INFRASTRUCTURE
│       │       ├── Help desk, device management? → OUTSOURCE
│       │       │   Cost: MSP $100-200/user/mo vs internal $60-90K/yr per tech
│       │       ├── Security operations? → OUTSOURCE (MSSP) unless >500 employees
│       │       └── Cloud architecture, DevOps? → IN-HOUSE if tech company, else OUTSOURCE
├── OVERRIDE CONDITIONS:
│   ├── Regulatory requirement mandates internal control → IN-HOUSE regardless
│   ├── <20 employees → Outsource everything except core product and sales
│   └── IP sensitivity is existential → IN-HOUSE regardless of cost
└── DEFAULT (if inputs are ambiguous):
    └── RECOMMEND: Start with outsourcing, build internal capability when volume justifies it
        Reason: Outsourcing is reversible; premature hiring is expensive and slow to undo
```

## Decision Logic

<!-- If/then rules for agents. These are the machine-readable version of
     the decision tree — agents can evaluate these programmatically. -->

### If function is core differentiator AND company has >50 employees
→ **In-House**. Core capabilities that create competitive advantage should never be outsourced. The 2-3x rebuilding cost for atrophied capabilities makes this effectively irreversible. [src2]

### If function is commodity AND company has <150 employees
→ **Outsource**. Small companies get disproportionate value from outsourcing commodity functions — a 50-person company cannot justify a full-time payroll specialist, benefits administrator, and compliance officer. [src6]

### If function is engineering AND company is a tech company
→ **Hybrid**. Keep architecture and product direction in-house (2-5 senior engineers). Augment with outsourced development capacity for feature execution. Internal senior engineers cost $150-200K/yr fully loaded; offshore senior developers cost $35-80K/yr. [src1]

### If function is customer support AND ticket volume >50/day
→ **Outsource Tier 1, retain Tier 2+3**. Outsourced Tier 1 costs $2-8/ticket vs $15-25/ticket internal. Keep escalation and customer success in-house to maintain brand quality. [src5]

### If regulatory or compliance requirement exists
→ **In-House or specialized firm**. Financial controls, SOX compliance, healthcare data, and privacy-regulated functions require either internal teams or specialized compliance-certified providers. General BPOs are insufficient. [src2]

### Default recommendation
→ **Outsource commodity functions first, build in-house when volume and strategic importance justify it**. The reversibility advantage of outsourcing means mistakes are cheap to fix. Premature hiring is 3-6x more expensive to unwind. [src3]

## Anti-Patterns

### Wrong: Outsourcing engineering to "move faster" without an internal architect
A startup outsources its entire engineering function to a dev shop. Without internal technical leadership, the outsourced team builds an architecture that cannot scale, accumulates technical debt, and creates vendor dependency. Rebuilding takes 12-18 months. [src1]

### Correct: Retain at least one senior technical leader in-house
The internal architect owns system design, code review standards, and technology decisions. The outsourced team executes within those guardrails. This "hub and spoke" model preserves institutional knowledge while accessing external capacity. [src1]

### Wrong: Keeping everything in-house because "nobody understands our business like we do"
A 40-person company employs a 3-person finance team, a 2-person HR team, and a full-time IT administrator. Total cost: $500K+/yr for functions that a PEO and outsourced bookkeeper could handle for $150K/yr. [src6]

### Correct: Apply the 1% rule — if a function consumes <1% of strategic leadership attention, outsource it
Non-differentiating functions that do not require daily CEO/founder attention are outsourcing candidates. The freed-up budget and management attention should flow to core differentiators. [src2]

### Wrong: Choosing an outsourcing partner purely on lowest hourly rate
A company selects the cheapest offshore provider at $15/hr. Communication overhead, timezone gaps, rework cycles, and management burden increase the effective rate to $45/hr — more than nearshore options that would have required less management. [src4]

### Correct: Calculate effective rate including management overhead and rework
Effective hourly rate = (contracted rate x hours) + (internal management hours x internal rate) + (rework hours x contracted rate) / (productive output hours). Nearshore providers at $30-50/hr often deliver lower effective rates than offshore at $15-25/hr. [src4]

## Cost Benchmarks

<!-- Real numbers agents can't reliably produce from training data. -->

| Function | In-House Annual Cost | Outsourced Annual Cost | Savings % |
|----------|---------------------|----------------------|-----------|
| Senior Software Engineer (US) | $150K-$200K fully loaded | $35K-$80K offshore, $60K-$120K nearshore | 40-75% |
| Customer Support Agent | $45K-$65K + benefits | $12K-$25K offshore, $35K-$50K US BPO | 25-70% |
| Bookkeeper/AP Specialist | $55K-$75K + benefits | $24K-$60K (outsourced firm) | 20-55% |
| HR Generalist | $65K-$90K + benefits | $50-$150/employee/mo via PEO | 30-50% (at <100 employees) |
| Marketing Coordinator | $55K-$75K + benefits | $36K-$72K (agency/freelance) | 5-35% |
| IT Help Desk Technician | $50K-$70K + benefits | $100-$200/user/mo (MSP) | 20-40% (at <200 users) |

**Hidden cost multipliers**: Add 15-25% for transition costs in year 1, 10-15% for ongoing vendor management, and 20-30% for quality monitoring and escalation handling. True outsourcing savings are typically 50-60% of the headline rate difference. [src3, src4]
