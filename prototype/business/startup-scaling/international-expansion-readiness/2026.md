# International Expansion Readiness

## Purpose

This recipe produces a market prioritization scorecard, localization depth matrix, entity formation decision guide, and international hiring plan — the four deliverables a startup founder needs to execute international expansion without wasting 6-12 months on the wrong market. The output tells you which market to enter first, how deep to localize, whether to use EOR or form an entity, and what it will cost for the first 12 months. [src1]

## Prerequisites
<!-- Agents: verify ALL prerequisites before executing. Missing prerequisites = failed execution. -->

- [ ] **Domestic product-market fit validated** — NPS > 30, retention > 80%, or equivalent evidence of repeatable sales [src1]
- [ ] **Revenue baseline** — minimum $500K ARR or clear path to sustainability (international expansion without revenue is premature)
- [ ] **Product i18n readiness** — codebase supports multi-language, multi-currency, or has clear path to support (estimate 2-6 weeks if not)
- [ ] **Legal counsel access** — corporate attorney familiar with international structuring (IP, tax, employment law)
- [ ] **Financial runway** — minimum 12 months runway to absorb expansion costs before expecting ROI

## Constraints
<!-- Hard rules. Agents: enforce throughout execution. Violating these = broken output or legal risk. -->

- Validate domestically before expanding — international expansion multiplies operational complexity 3-5x [src1]
- Entity formation costs $5K-50K per jurisdiction and takes 2-12 weeks — do not underestimate setup time [src2]
- EOR services cost $400-700/employee/month on top of salary — switch to own entity above 10-15 employees per country [src5]
- Localization requires cultural adaptation beyond translation — pricing, payment methods, support hours, legal compliance [src3]
- Tax nexus triggers vary by country — consult local tax advisor before any hiring or commercial activity

## Tool Selection Decision

```
What is the expansion driver?
├── Access new customers (revenue growth)
│   ├── Team size < 5 in new market
│   │   └── PATH A: EOR + Light Localization
│   └── Team size > 5 in new market
│       └── PATH B: Entity Formation + Full Localization
├── Talent arbitrage (cost reduction)
│   ├── Engineering/product roles only
│   │   └── PATH C: EOR + No Product Localization
│   └── Customer-facing roles
│       └── PATH D: EOR → Entity (hybrid approach)
└── Follow existing customers
    └── PATH B: Entity Formation + Full Localization
```

| Path | Model | Setup Cost | Monthly Cost/Employee | Timeline | Best For |
|------|-------|------------|----------------------|----------|----------|
| A: EOR + Light | Deel/Remote | $0-2K | $400-700 + salary | 2-4 weeks | Testing a market with < 5 people |
| B: Entity + Full | Local incorporation | $10K-50K | Salary + 20-40% employer costs | 2-4 months | Committed market entry with 5+ people |
| C: EOR Only | Deel/Remote | $0-2K | $400-700 + salary | 1-2 weeks | Remote hiring for cost, no market entry |
| D: EOR → Entity | Start EOR, convert later | $0-2K initially | $400-700 initially | 1-2 weeks start | Validate before committing to entity |

## Execution Flow

### Step 1: Market Prioritization Scoring

**Duration**: 3-5 days
**Tool**: Spreadsheet + market research (Google Market Finder, Statista, SimilarWeb)

Score each candidate market across 8 dimensions using the MARACA-inspired framework. [src3] [src4]

```
MARKET PRIORITIZATION SCORECARD (score each 1-10):

| Dimension                | Weight | Market A | Market B | Market C |
|--------------------------|--------|----------|----------|----------|
| 1. Total addressable market (TAM) | 20% |          |          |          |
| 2. Customer readiness/demand      | 15% |          |          |          |
| 3. Competitive intensity          | 15% |          |          |          |
| 4. Localization complexity        | 15% |          |          |          |
| 5. Regulatory burden              | 10% |          |          |          |
| 6. Talent availability            | 10% |          |          |          |
| 7. Payment infrastructure         | 10% |          |          |          |
| 8. Cultural distance              | 5%  |          |          |          |
| WEIGHTED TOTAL                     | 100%|          |          |          |
```

**Data sources for each dimension**:
- TAM: Statista, Gartner, local industry reports
- Customer readiness: Google Trends (search volume in local language), SimilarWeb (competitor traffic from region)
- Competitive intensity: Crunchbase (funded competitors in market), G2/Capterra (local alternatives)
- Localization complexity: Language distance, payment method diversity, regulatory requirements
- Regulatory burden: World Bank Ease of Doing Business, local legal requirements
- Talent availability: LinkedIn Talent Insights, Glassdoor salary data
- Payment infrastructure: Stripe Atlas coverage, local payment preferences (iDEAL in NL, PIX in BR, UPI in IN)
- Cultural distance: Hofstede dimensions, timezone overlap with HQ

**Verify**: At least 3 markets scored with data-backed scores (not guesses). Top market has weighted score > 6.5/10.
**If failed**: If data is insufficient, run a 2-week demand test (targeted ads in each market, measure CPL and conversion rate) before scoring.

### Step 2: Localization Depth Assessment

**Duration**: 2-3 days
**Tool**: Product audit + market research

For the top-ranked market, determine the required depth of localization across five layers. [src3]

```
LOCALIZATION DEPTH MATRIX:

| Layer           | Minimum (MVP)                | Standard               | Deep                        |
|-----------------|------------------------------|------------------------|-----------------------------|
| Language        | English only + key UI strings | Full UI translation    | Full product + content + SEO |
| Pricing         | USD with local equivalent     | Local currency pricing | Localized pricing strategy   |
| Payment         | Stripe/PayPal                | + 1-2 local methods    | All major local methods      |
| Legal/Compliance| Privacy policy update         | Local T&Cs + DPA       | Full regulatory compliance   |
| Support         | English, HQ timezone          | Local language, expanded hours | Local team, local timezone |
| Marketing       | Translated landing page       | Localized content strategy | Local team, local channels  |
```

**Decision rule**: Start with Minimum for first 3 months, upgrade to Standard when you have 10+ paying customers in market, go Deep only when market contributes > 20% of revenue.

**Verify**: Localization requirements documented for top 2 markets, cost and timeline estimated for each layer
**If failed**: If product i18n is not ready, add 4-8 weeks and $10K-30K to timeline/budget for i18n implementation first

### Step 3: Entity Structure Decision

**Duration**: 2-3 days (research) + 1-2 weeks (legal consultation)
**Tool**: Legal advisor + EOR platform comparison

Decide between EOR, contractor, and entity formation for each market.

```
ENTITY DECISION FRAMEWORK:

Headcount in country?
├── 0 (selling remotely only)
│   └── No entity needed — confirm no tax nexus trigger
├── 1-3 employees
│   └── EOR (cost: $400-700/person/month + salary)
│       ├── Deel: $599/employee/month (global coverage)
│       ├── Remote: $599/employee/month (own entities in 80+ countries)
│       └── Oyster: $599/employee/month (90+ countries)
├── 4-10 employees
│   └── EOR initially, plan entity formation when > 10 (12-18 month horizon)
├── 11-20 employees
│   └── Form entity — EOR costs exceed entity overhead at this scale
│       ├── Simple markets (UK, Netherlands, Ireland): $5K-15K, 2-6 weeks
│       ├── Medium markets (Germany, France, Australia): $10K-25K, 4-8 weeks
│       └── Complex markets (Brazil, India, China): $20K-50K, 8-12 weeks
└── 20+ employees
    └── Entity required — engage Big 4 or specialized firm for structuring
```

**Cost comparison (annual, 10 employees at $80K avg salary)**:

| Model | Year 1 Cost | Year 2 Cost | Year 3 Cost |
|-------|-------------|-------------|-------------|
| EOR (Deel) | $871K ($800K salary + $71K EOR fees) | $871K | $871K |
| Own entity (simple market) | $870K ($800K + $55K employer costs + $15K setup) | $855K | $855K |
| Own entity (complex market) | $920K ($800K + $80K employer + $40K setup) | $880K | $880K |

**Verify**: Entity decision made for top market with cost projection for 3 years, legal advisor consulted
**If failed**: If unsure about tax implications, engage international tax advisor (budget $3K-10K)

### Step 5: Financial Model and Go/No-Go Decision

**Duration**: 2-3 days
**Tool**: Spreadsheet (financial model)

Build a 24-month financial model for the expansion that answers: what does the expansion cost, when does it break even, and what is the risk if it fails?

```
EXPANSION FINANCIAL MODEL TEMPLATE:

| Line Item                  | Month 1-3    | Month 4-6    | Month 7-12   | Year 2       |
|----------------------------|-------------|-------------|-------------|-------------|
| Setup costs (entity/legal)  | $5K-50K     | —           | —           | —           |
| Hiring costs (recruiting)   | $5K-15K     | $3K-8K      | $2K-5K      | $2K-5K/qtr  |
| Salaries + employer costs   | $X/month    | $X/month    | $X/month    | $X/month    |
| EOR fees (if applicable)    | $Y/month    | $Y/month    | —           | —           |
| Office/coworking            | $0-5K/mo    | $0-5K/mo    | $0-8K/mo    | $0-8K/mo    |
| Localization                | $5K-20K     | $2K-5K      | $1K-3K      | $1K-3K/qtr  |
| Marketing (local)           | $2K-10K/mo  | $5K-20K/mo  | $5K-20K/mo  | $5K-30K/mo  |
| Legal/compliance ongoing    | $1K-3K/mo   | $1K-3K/mo   | $1K-3K/mo   | $1K-3K/mo   |
| TOTAL COST                  |             |             |             |             |
| Revenue from new market     | $0-5K       | $5K-20K     | $20K-100K   | $100K-500K+ |
| NET                         |             |             |             |             |
```

**Go/No-Go criteria**:
- Go: Payback within 18 months, market scores > 6.5/10, total investment < 6 months runway
- Conditional Go: Payback 18-24 months, market scores 5.5-6.5/10 — proceed with smaller team, faster kill switch
- No-Go: Payback > 24 months or total investment > 6 months runway or market scores < 5.5/10

**Verify**: Model completed with realistic assumptions, sensitivity analysis on 3 variables (time to first deal, salary costs, marketing spend)
**If failed**: If revenue projections are pure speculation, run a 30-day market test ($2K-5K in ads, measure qualified leads) before building full model

### Step 6: Execution Kickoff and 90-Day Plan

**Duration**: 1-2 days (planning) + 90 days (execution)
**Tool**: Project management tool + EOR platform

```

├── Engage local legal counsel or EOR
├── Begin entity formation (if PATH B) or EOR onboarding (if PATH A/C/D)
├── Register for local tax obligations
└── Set up local bank account (if entity)

Week 3-4: Product localization sprint
├── Implement minimum localization layer
├── Add local payment method(s)
├── Update privacy policy and T&Cs for local jurisdiction
└── Set up local customer support channel

Week 5-8: First hires and market activation
├── Hire first 1-3 employees via EOR or entity
├── Launch localized landing page and marketing
├── Begin outbound sales / inbound lead generation
└── Establish local partnerships if applicable

Week 9-12: Validate and adjust
├── Review first customer conversations / deals
├── Measure: CPA, conversion rate, deal size vs domestic
├── Decision: double down, adjust, or exit
└── Report to board with 6-month projection
```

## Error Handling

| Error | Likely Cause | Recovery Action |
|-------|-------------|----------------|
| Entity formation delayed beyond 8 weeks | Local bureaucracy or missing documentation | Use EOR as interim solution, continue entity process in parallel |
| Salary benchmarks significantly outdated | Rapid market changes or limited data | Cross-reference with 3+ sources, request live quotes from EOR |

| Product localization takes > 6 weeks | Technical debt in i18n layer | Reduce localization scope to language + currency only, defer deeper adaptation |

## Anti-Patterns

### Wrong: Boil the Ocean Localization
Fully translating the entire product, help docs, and marketing site before acquiring a single customer in the new market. This wastes $20K-50K if the market turns out to be wrong. [src3]

### Correct: Minimum Viable Localization
Launch with translated UI, local payment methods, and localized landing page. Deepen localization only after validating demand with first 10-20 customers. Revenue justifies investment.

### Wrong: Forming Entity Immediately
Spending $15K-50K and 2-3 months on entity formation before validating whether the market will generate revenue. Dissolution costs another $5K-15K if the expansion fails.

### Correct: EOR-First, Entity-When-Proven
Start with EOR for first 3-6 employees. Convert to own entity only when headcount exceeds 10-15 (cost crossover point) or when entity provides strategic advantages (government contracts, local banking). [src5]

### Wrong: Assuming Domestic Playbook Transfers
Copying exact sales motions, pricing, and marketing channels from domestic market to international market. B2B sales cycles, buying committees, and decision processes vary significantly by culture. [src1]

### Correct: Localization Discovery Process
Conduct 10-15 interviews with potential customers and partners in the target market before committing to a go-to-market strategy. Discover what changes about the buying process, not just the language. [src3]
