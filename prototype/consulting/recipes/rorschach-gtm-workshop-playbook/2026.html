How do you run a 2-day Rorschach GTM workshop with 8 modules?

Type: Execution Recipe Confidence: 0.85 Sources: 5 Verified: 2026-03-30

Purpose

Full 2-day workshop execution guide transforming a client's go-to-market from volume-based outbound to a precision system built on ambiguous signal design, exhaust fume detection, intentional friction, buying committee dynamics, counterfactual inoculation, and organizational immune navigation. Revenue: $12K workshop + $3K/month advisory retainer. [src1, src2]

Prerequisites

  • Client Industry Profile — from pre-workshop intake
  • Participant List — minimum 3 from client side with roles
  • Pipeline Metrics — inquiry volume, qualification rate, close rate, deal size
  • Target Accounts — 3-5 accounts with firmographic data
  • Facilitator prepared — all knowledge cards reviewed, materials ready

Constraints

  • All 8 modules in sequence — later modules depend on earlier outputs [src2]
  • 2 hours per module minimum — compressing below 90 min degrades output quality
  • Minimum 3 client participants — fewer limits exercise diversity
  • Friction gates must demonstrate Costly Signaling Theory [src3]
  • 90-day milestones must be specific and measurable [src5]

Execution Flow

Day 1: Foundation & Infrastructure

Module 1 (2h): Rorschach Signal Design — identify failure states, craft ambiguous artifacts, validate with healthy company test. Verify: 3-5 artifacts per segment.

Module 2 (2h): Category Redefinition — map competitive landscape, identify gap, design new category. Verify: Category manifesto in uncontested territory.

Module 3 (2h): Exhaust Detection — inventory signal sources, design compound triggers, specify tool stack. Verify: 3+ compound triggers with thresholds.

Module 4 (2h): Friction Gate Architecture — design diagnostic tools, multi-stakeholder workshops, calculators. Verify: 3+ gates with filter rate projections.

Day 2: Dynamics & Integration

Module 5 (2h): Committee Topology — map stakeholder roles, design waveforms, build silence protocol. Verify: Committee model for 3+ account types.

Module 6 (2h): Counterfactual Modeling — build entropy trajectories, mock incident reports. Verify: 3 account-specific packages.

Module 7 (2h): Immune Navigation — identify antibodies, design circumvention strategies. Verify: Constraint maps for target accounts.

Module 8 (2h): Integration & Roadmap — cross-reference outputs, build 90-day plan, scope retainer. Verify: Complete roadmap with owners and metrics.

Anti-Patterns

Wrong: Running modules out of sequence

Friction gates without Rorschach signal context look like traditional lead qualification.

Correct: Follow the 8-module sequence

Each module's output feeds into later modules, building from theory to infrastructure to dynamics to integration.

Wrong: Compressing into a single day

Cutting exercises means participants understand theory but cannot apply it. [src1]

Correct: Maintain 2-hour module blocks with exercise time

Exercises are where participants internalize frameworks and produce client-specific outputs.

When This Matters

Use when delivering a full Rorschach GTM engagement to a client wanting to transform from volume-based outbound to precision go-to-market. Requires 2 full days, minimum 3 participants, produces comprehensive deliverable with 90-day roadmap.
